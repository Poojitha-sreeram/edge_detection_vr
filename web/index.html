About This Viewer

This is a minimal web viewer demonstrating the integration of Android native processing with a web-based display layer. The processed frame from the Android application (edge detection output from OpenCV) is displayed here with real-time statistics.

Features:

  • Real-time frame display from native processing
  • FPS and performance metrics
  • Processing mode toggle
  • Sample frame loading for demonstration
  • TypeScript-based modular architecture
